Deploymet tools for figureskate-judging-deployment
===================================================

ABOUT
-----
This repository supplies below tools.  
* Make easy development environment
* Provisioning
* Deployment

SERVER ENVIRONMENT
-------------------
* CentOS
* Ubuntu

USAGE
-----
```bash
cd ubuntu(or centos)
# Provisioning and Deploymet
fab setup_devenv
# Deployment
fab deploy
```
